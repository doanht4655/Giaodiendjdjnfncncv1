## 🚀 Cài Đặt & Chạy

### Yêu Cầu
- Python 3.6+
- Internet connection

### 🔧 Cài Đặt Tự Động (Khuyến nghị)

```bash
# Tải về và setup tự động
git clone https://github.com/doanht4655/Giaodiendjdjnfncncv1.git
cd Giaodiendjdjnfncncv1
python3 setup.py
```

### 🏃‍♂️ Chạy Nhanh

**Linux/Mac:**
```bash
# Cách 1: Script shell
./run.sh

# Cách 2: Makefile
make run

# Cách 3: Trực tiếp
python3 main_Version2.py
```

**Windows:**
```cmd
# Cách 1: Batch script
run.bat

# Cách 2: Trực tiếp
python main_Version2.py
```
